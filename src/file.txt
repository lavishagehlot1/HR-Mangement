# Simple Task Management API (RBAC Learning Project)

Goal:
Learn how **different roles access different endpoints**.

Roles:

* **Admin**
* **Manager**
* **Employee**

---

# 1. Features (Keep it Minimal)

You only need **2 resources**:

### Users

Stores system users.

### Tasks

Stores tasks assigned to employees.

# 5. Simple RBAC Middleware

This is the **main part of RBAC**.

```javascript
function authorize(role) {

  return (req, res, next) => {

    if (req.user.role !== role) {
      return res.status(403).json({
        message: "Access denied"
      });
    }

    next();

  };

}
```

Example usage:

```
POST /users → admin only
POST /tasks → manager only
DELETE /tasks → admin only
```

Example:

```javascript
router.post("/tasks",
  authenticate,
  authorize("manager"),
  createTask
);
```

---

# 6. Authentication Middleware

Verify token using **JSON Web Token**.

```javascript
function authenticate(req, res, next) {

  const token = req.headers.authorization;

  if (!token) {
    return res.status(401).json({ message: "No token" });
  }

  const decoded = jwt.verify(token, "secret");

  req.user = decoded;

  next();
}
```
